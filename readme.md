## Run Locally

Clone the project

```bash
  git clone https://github.com/prashantxi786/frantic-week-8502.git
```

Go to the project directory

```bash
  cd frantic-week-8502
```

Install dependencies for frontend as well as backend

```bash
  npm install
```


Start the localhost server

```bash
  npm start
```
